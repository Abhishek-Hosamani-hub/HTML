HTML

Introduction

HTML stands for Hyper Text Markup Language. It is used to create web pages. It is the backbone of the web. We can use HTML to create headings, paragraphs, links, images, forms, tables, lists, etc. We can say it creates skeleton of any web page.

History of HTML

HTMl was first introduced in 1989 by Tim Berners-Lee in the year 1992. First HTML based web page was created in 1992
